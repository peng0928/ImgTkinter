## 📖 项目概述

**ImgTkinter** 是一个基于 PyQt5 开发的现代化桌面应用程序，专为高效管理、分类和组织大量图片文件而设计。它提供了直观的图形界面，支持批量操作、智能搜索和灵活的图片管理功能，帮助用户轻松整理海量图片资源。

## ✨ 核心特性

### 🎨 现代化界面设计
- **紫色主题界面** (#8B5CF6)，美观大方
- **双面板布局**：左侧图片浏览区 + 右侧文件夹管理区
- **响应式设计**，自适应窗口大小
- **平滑的动画和交互效果**

### 📷 强大的图片管理功能
- **支持多种格式**：PNG, JPG, JPEG, GIF, BMP, TIFF, WEBP
- **分页显示**：每页最多显示200张图片，支持海量图片加载
- **多选操作**：Shift+单击选择多个图片
- **右键快捷菜单**：删除、重命名、查看、复制路径
- **图片预览**：双击图片放大查看，支持缩放操作

### 📁 智能文件夹管理
- **自动图标生成**：使用文件夹内第一张图片作为图标
- **文件数量可视化**：不同数量显示不同颜色标记
- **拖拽操作**：直接拖拽图片到目标文件夹进行分类
- **批量创建和管理**：快速创建多个分类文件夹

### 🔍 高效的搜索与排序
- **智能搜索**：支持中文拼音搜索（如输入"tp"可匹配"图片"）
- **数量排序**：按文件夹内图片数量升序/降序排列
- **快速聚焦**：空格键快速跳转到搜索框
- **实时筛选**：输入时实时显示匹配结果

### ⚡ 性能优化
- **异步加载**：多线程加载图片，界面不卡顿
- **智能缓存**：记忆用户操作和设置
- **批量操作**：支持批量移动、删除图片
- **错误处理**：完善的异常捕获和用户提示
